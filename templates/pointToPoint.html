Access points


These devices allow you to connect two access points. It is understood by Multipoint to the ability to make connections with several nodes of a network.

Our router communicates with two access points. Before setting up the Multipoint router, you have to enable the other two points as a bridge and point-to-point towards the router.

Once finished, the MAC address of the router and the other access points are copied using the address 192.168.1.1 in the browser of your choice. Then the LAN IP of the router is changed, the changes are saved, restart and return to enter with the new IP address.

To change the router to a multipoint configuration, within the router's web application look for the Wireless option or similar (it depends on the router model) we locate the Point to multi-Point option, then mark or select in the box Mac addresses, you enter the access point addresses which you want to communicate with the Point to Multipoint configuration.



OSPF Point-to-Point Networks


This image defines 5 types of router

Point to Point: Two routers connected through a link (WAN).

Broadcast Multiaccess: Several routers connected to an Ethernet network

Non-Broadcast Multi-Access (NMBA): Several routers interconnected in a network that does not allow broadcast transmission.

Point to Multipoint: Several routers interconnected in a hub-and-spoke topology through an NBMA network.

Virtual Links: An OSPF network is used to interconnect different areas and backbone.
